# fscan GUI - 内网综合扫描工具图形界面

fscan GUI是基于PyQt5开发的fscan图形界面工具，将fscan的命令行功能可视化，提供更友好的操作体验。

## 功能特点

### 1. 完整的fscan功能支持
- 支持fscan所有命令行参数
- 图形化界面，无需记忆复杂命令
- 实时显示扫描结果

### 2. 分类清晰的功能模块
- **主机扫描**：整合目标配置、扫描控制、认证与暴力破解
- **Web扫描**：包含Web目标、Web设置、POC测试、认证与暴力破解
- **系统漏洞利用**：Redis利用等非Web相关漏洞利用
- **输出与显示**：结果输出格式和显示设置
- **系统设置**：API设置、语言设置、工具路径设置

### 3. 增强的用户体验
- 扫描模式支持复选框和全选功能
- 扫描模式分类：网络服务、数据库服务、消息队列、漏洞检测
- 认证和暴力破解功能在主机和Web扫描中均可用
- 支持文件浏览选择弱口令字典
- 实时扫描结果显示和搜索功能

## 安装和使用

### 方式一：直接运行Python脚本
1. 确保安装了Python 3.6+和PyQt5
2. 安装依赖：
   ```bash
   pip install PyQt5
   ```
3. 运行GUI：
   ```bash
   python fscan_gui.py
   ```

### 方式二：使用打包好的可执行文件
1. 直接运行`dist/fscan_gui.exe`（Windows平台）
2. 确保fscan可执行文件与GUI在同一目录，或在系统设置中指定路径

## 主要功能模块说明

### 主机扫描
- **目标地址**：设置扫描目标、排除主机、端口配置等
- **扫描模式**：选择要扫描的服务类型，支持全选/取消全选
- **性能设置**：调整扫描线程数、超时时间等
- **存活探测**：Ping设置、指纹识别等
- **认证与暴力破解**：基本认证、弱口令字典、Hash认证、高级认证

### Web扫描
- **Web目标**：设置目标URL、URL文件、Cookie等
- **Web设置**：Web超时时间、代理设置等
- **Web POC测试**：POC路径、POC名称、完整扫描等
- **认证与暴力破解**：与主机扫描共享相同的认证设置界面

### 输出与显示
- **输出设置**：输出文件、输出格式、禁用结果保存等
- **显示设置**：静默模式、彩色输出、日志级别等

## 打包说明

如果需要自行打包，可以使用PyInstaller：

1. 安装PyInstaller：
   ```bash
   pip install pyinstaller
   ```

2. 打包命令：
   ```bash
   pyinstaller -F -w --name=fscan_gui fscan_gui.py
   ```

3. 打包结果：
   - 生成的可执行文件位于`dist/fscan_gui.exe`
   - `-F`：生成单个可执行文件
   - `-w`：不显示控制台窗口

## 注意事项

1. 确保fscan可执行文件可正常运行
2. 扫描过程中请勿关闭窗口，否则会终止扫描
3. 对于大规模扫描，建议调整线程数以避免系统资源占用过高
4. 暴力破解功能请在合法授权下使用
